# Metalworking-Zombies-Project-Zomboid---B42
A highly customizable B42 Project Zomboid mod. Allows a rare metalworking zombie to spawn in the wild, allows for a renewable source of metal in the late game.

This mod has sandbox options that includes:
1. The ability to change how rare/common the metalworking zombie should spawn.
2. Different types of metal, and independent spawn rate on said metalworking zomboid's backpack.
3. How many items can spawn 0-30.
